MicroService Base
-----------------

```
git clone git@bitbucket.org:orbisunt/aptitus-microservice-base.git
```

1 cambiar el nombre al proyecto, make file line:5
```
SERVICE_NAME 	= base 
```
```
SERVICE_NAME 	= otro_nombre 
```

iniciar el proyecto
```
make up
```

log
```
make log
```

test unitarios
```
make tests
```

test e2e
```
make tests-e2e
```
